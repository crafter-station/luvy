# Triage Labels

This repo uses the default triage label vocabulary.

- `needs-triage`: maintainer needs to evaluate the issue or PRD.
- `needs-info`: waiting on the reporter or requester for more information.
- `ready-for-agent`: fully specified and ready for an AFK agent to implement.
- `ready-for-human`: needs human implementation or human judgment before implementation.
- `wontfix`: will not be actioned.

New PRDs should start with `needs-triage` unless the user explicitly says otherwise.
